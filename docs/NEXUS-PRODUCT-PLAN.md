**产品愿景**: 让每个电商卖家都能 30 秒生成专业级产品 3D 展示

**核心价值**:

### 2.2 核心技术方案

#### 方案 1: 材质语义对齐

**技术栈**:
```typescript
// lib/rag/material-params.ts

import { QdrantClient } from '@qdrant/js-client-rest'
import { OpenAI } from 'openai'

interface MaterialParams {
  roughness: number
  metalness: number
  clearcoat?: number
  anisotropy?: number
}

async function inferMaterialParams(description: string): Promise<MaterialParams> {
  // 1. 检索材质知识库（混合检索：BM25 + 向量）
  const knowledge = await ragSearch(description, 'material_params', {
    topK: 5,
    useRerank: true,
  })
  
  // 2. LLM 推理（处理模糊语义）
  const llm = new OpenAI({ apiKey: process.env.DASHSCOPE_API_KEY })
  const response = await llm.chat.completions.create({
    model: 'qwen3.5-plus',
    messages: [{
      role: 'user',
      content: `
        用户描述：${description}
        参考知识库：${JSON.stringify(knowledge)}
        
        请输出 PBR 参数（roughness/metalness/clearcoat/anisotropy）
        返回 JSON 格式
      `,
    }],
    response_format: { type: 'json_object' },
  })
  
  // 3. 融合结果
  return mergeParams(knowledge, JSON.parse(response.choices[0].message.content))
}
```

**知识库结构**:
```json
{
  "id": "material-001",
  "name": "丝滑",
  "category": "触感",
  "params": {
    "roughness": 0.2,
    "metalness": 0.1,
    "clearcoat": 0.8
  },
  "examples": ["丝绸", "缎面", "高档护肤品瓶身"],
  "synonyms": ["顺滑", "柔滑", "光滑"]
}
```

**验收标准**:
- 材质覆盖度：50+ 条目
- 参数推理准确率：>85%
- 检索延迟：<500ms

#### 方案 2: 3D 生成质量优化

**问题**: Tripo 生成质量粗糙

**解决方案**: 三层优化

```typescript
// lib/enhance/tripo-post-process.ts

async function enhanceTripoModel(model: GLTF): Promise<GLTF> {
  // 第 1 层：网格优化
  const smoothed = await smoothMesh(model, {
    iterations: 3,
    strength: 0.5,
    preserveEdges: true,
  })
  
  // 第 2 层：材质增强
  const enhanced = await enhanceMaterials(smoothed, {
    roughnessAdjust: -0.1,  // 更光滑
    metalnessAdjust: +0.1,  // 更有质感
    normalMapStrength: 0.5,
  })
  
  // 第 3 层：高质量渲染
  const rendered = await render(enhanced, {
    samples: 128,  // 高质量采样
    hdr: 'studio_lighting.hdr',
    toneMapping: 'aces',
    shadowMap: true,
    ambientOcclusion: true,
  })
  
  return rendered
}
```

**效果对比**:
| 优化层 | 质量提升 | 耗时增加 |
|--------|----------|----------|
| Tripo 原始 | ⭐⭐⭐ | 30 秒 |
| + 网格优化 | ⭐⭐⭐⭐ | +5 秒 |
| + 材质增强 | ⭐⭐⭐⭐ | +3 秒 |
| + 高质量渲染 | ⭐⭐⭐⭐⭐ | +10 秒 |

#### 方案 3: 多平台尺寸适配

**技术实现**:
```typescript
// lib/export/platform-adapter.ts

interface PlatformSpec {
  width: number
  height: number
  aspectRatio: string
  maxSizeMb: number
  requirements: string[]
}

const PLATFORM_SPECS: Record<string, PlatformSpec> = {
  taobao: {
    width: 800,
    height: 800,
    aspectRatio: '1:1',
    maxSizeMb: 3,
    requirements: ['白底或纯色背景', '产品占比>70%', '无文字水印'],
  },
  xiaohongshu: {
    width: 1242,
    height: 1660,
    aspectRatio: '3:4',
    maxSizeMb: 10,
    requirements: ['竖图优先', '可加文字/贴纸'],
  },
  douyin: {
    width: 1080,
    height: 1920,
    aspectRatio: '9:16',
    maxSizeMb: 5,
    requirements: ['竖图', '适合手机浏览'],
  },
}

async function adaptToPlatform(
  model: ThreeModel,
  platform: string
): Promise<ExportResult> {
  const spec = PLATFORM_SPECS[platform]
  
  // 1. 调整相机视角
  camera.aspect = spec.width / spec.height
  camera.updateProjectionMatrix()
  
  // 2. 调整灯光
  setupLightingForPlatform(platform)
  
  // 3. 渲染输出
  const image = await render(model, camera, spec)
  
  // 4. 压缩优化
  const optimized = await optimizeImage(image, spec.maxSizeMb)
  
  return optimized
}
```
