## 1. 设计原则

### 1.1 核心原则

| 原则 | 描述 | 应用场景 |
|------|------|----------|
| **Tech-Forward** | 拥抱未来感、AI 驱动的视觉语言 | 深色背景、霓虹光效、渐变 |
| **沉浸感** | 3D 产品需要 3D 视觉氛围 | 背景、过渡、交互反馈 |
| **简洁高效** | 30 秒完成核心流程 | 减少步骤、智能默认 |
| **动效即反馈** | 每个交互都有视觉回应 | 加载、完成、错误状态 |

### 1.2 视觉风格 (Tripo-Inspired)

```
品牌调性: 科技感 · 专业感 · 高端感
视觉关键词: 暗黑 · 霓虹 · 流动 · 粒子
设计灵感: Tripo3D - 暗黑背景 + 渐变文字 + 发光元素
```

## 3. 3D 动效背景规范

### 3.1 首页 Hero 背景

**效果**: 浮动的 3D 几何体 + 粒子流光

```
技术方案:
- Three.js + React Three Fiber
- 浮动几何体: 球体、立方体、环形，随机运动
- 粒子系统: 100-200 个粒子，流光效果
- 鼠标交互: 视差效果，几何体跟随鼠标缓动
- 性能: 限制 60fps，移动端降级为静态图
```

**视觉参数**:
```typescript
// 几何体配置
const geometryConfig = {
  count: 8,
  size: { min: 0.5, max: 2 },
  speed: { min: 0.2, max: 0.5 },
  rotation: true,
  floating: true
}

// 粒子配置
const particleConfig = {
  count: 150,
  size: 2,
  color: '#3B82F6',
  opacity: 0.6,
  speed: 0.1
}
```

### 3.2 生成页面背景

**效果**: 网格地平线 + 光晕

```
技术方案:
- CSS Grid 渐变背景
- 中央光晕效果 (radial-gradient)
- 动态波纹动画 (CSS animation)
```

### 3.3 Agent 控制台背景

**效果**: 代码雨 / 数据流

```
技术方案:
- Canvas 动画
- 下落的代码字符 (0-9, A-F)
- 绿色主色调 (Matrix 风格)
- 速度可调节
```

## 4. 组件动效规范

### 4.1 入场动画

| 组件 | 动画 | 时长 | 缓动函数 |
|------|------|------|----------|
| 卡片 | `fade-up` (淡入上移) | 400ms | `ease-out` |
| 按钮 | `scale` (缩放弹入) | 200ms | `ease-out` |
| 标题 | `fade` (淡入) | 300ms | `ease-out` |
| 图片 | `blur` (模糊到清晰) | 500ms | `ease-out` |

### 4.2 交互动画

| 交互 | 动画 | 时长 |
|------|------|------|
| 按钮悬停 | `scale(1.02)` + 阴影增强 | 150ms |
| 按钮点击 | `scale(0.98)` | 100ms |
| 卡片悬停 | `translateY(-4px)` + 阴影增强 | 200ms |
| 输入聚焦 | 边框颜色渐变 | 150ms |
| Tab 切换 | 滑动指示器 | 200ms |

### 4.3 加载状态

| 状态 | 动画 | 说明 |
|------|------|------|
| 上传中 | 环形进度条 + 脉冲光效 | 显示进度百分比 |
| 生成中 | 骨架屏 + 波纹扩散 | 提示 "AI 正在工作" |
| 完成 | 成功图标弹入 + 粒子庆祝 | 持续 1s 后消失 |

## 7. 性能要求

| 指标 | 目标值 |
|------|--------|
| 首屏加载 (FCP) | < 1.5s |
| 可交互时间 (TTI) | < 3s |
| 动画帧率 | ≥ 60fps |
| 3D 背景内存 | < 50MB |

**优化策略**:
1. 3D 背景使用 `requestAnimationFrame`
2. 移动端降级为静态图片
3. 懒加载非首屏动效
4. 使用 CSS `will-change` 优化动画

### 11.2 CSS 代码示例

```css
/* 霓虹发光效果 */
.glow-blue {
  box-shadow: 0 0 20px rgba(0, 212, 255, 0.3),
              0 0 40px rgba(0, 212, 255, 0.1);
}

/* 卡片悬停抬起 */
.card-hover {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}
.card-hover:hover {
  transform: translateY(-4px);
  box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);
}

/* 渐变文字 */
.gradient-text {
  background: linear-gradient(135deg, #00D4FF, #8B5CF6, #F472B6);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

/* 玻璃态导航 */
.glass-nav {
  background: rgba(0, 0, 0, 0.8);
  backdrop-filter: blur(12px);
  border-bottom: 1px solid rgba(255, 255, 255, 0.05);
}

/* 脉冲动画 */
@keyframes pulse-glow {
  0%, 100% { opacity: 0.5; }
  50% { opacity: 1; }
}

/* 无限滚动 */
@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
```

### 11.3 Framer Motion 动画示例

```tsx
// 字符逐个淡入
const textVariants = {
  hidden: { opacity: 0, y: 20 },
  visible: (i: number) => ({
    opacity: 1,
    y: 0,
    transition: { delay: i * 0.05, duration: 0.4 }
  })
}

// 数字翻转
const counterVariants = {
  initial: { y: 0 },
  animate: (value: number) => ({
    y: -value * 100,
    transition: { duration: 1.5, ease: 'easeOut' }
  })
}

// 卡片入场
const cardVariants = {
  hidden: { opacity: 0, y: 30 },
  visible: {
    opacity: 1,
    y: 0,
    transition: { duration: 0.5, ease: [0.16, 1, 0.3, 1] }
  }
}
```
