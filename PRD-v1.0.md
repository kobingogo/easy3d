## 一、项目概述

### 1.1 产品定位
**easy3d** 是一款面向电商卖家的 AI 3D 展示生成工具，让卖家无需专业建模技能，即可将商品图片快速转换为 3D 展示内容。

**Slogan**: 一张图，30 秒，生成你的 3D 商品展示

### 1.2 核心价值
| 维度 | 传统方案 | easy3d |
|------|----------|--------|
| 成本 | ¥500-2000/个 | ¥99/月无限生成 |
| 时间 | 2-5 天/个 | 30 秒/个 |
| 门槛 | 专业建模师 | 零基础上手 |
| 批量 | 手动逐个 | 一次 100+ |

### 1.3 目标用户
**核心用户**: 淘宝/拼多多/抖音中小卖家
- 店铺规模：1-50 个 SKU
- 年营收：50-500 万
- 痛点：主图同质化严重，转化率低
- 特征：对价格敏感，追求 ROI

**次要用户**: 跨境电商卖家 (亚马逊/Temu/Shein)
- 需求：AR 展示、多平台分发
- 支付能力更强

## 二、功能需求 (v1.0 MVP)

### 2.1 核心功能

#### F1: 图片上传
| 字段 | 说明 |
|------|------|
| 支持格式 | JPG, PNG, WebP |
| 单张大小 | ≤10MB |
| 批量上限 | 10 张/次 (v1.0) |
| 自动处理 | 智能去背、裁剪、增强 |

#### F2: 3D 模型生成
| 字段 | 说明 |
|------|------|
| 生成引擎 | Tripo AI API (初期) |
| 生成时间 | 30-60 秒/个 |
| 输出格式 | GLB (Web 友好) |
| 质量选项 | 标准 (默认) / 高清 |

#### F3: 3D 预览播放器
| 字段 | 说明 |
|------|------|
| 技术栈 | Three.js + R3F |
| 交互 | 旋转、缩放、平移 |
| 光照 | 自动环境光 + 可选打光 |
| 背景 | 纯色/渐变/透明 |

#### F4: 导出与分享
| 字段 | 说明 |
|------|------|
| 导出格式 | GLB, GIF, MP4 (旋转展示) |
| 嵌入代码 | HTML iframe (一键复制) |
| 分享链接 | 临时预览链接 (7 天有效) |

### 2.2 辅助功能

#### F5: 用户系统
| 字段 | 说明 |
|------|------|
| 登录方式 | 微信扫码 (优先) / 手机号 |
| 免费额度 | 3 个/日 (引流) |
| 会员体系 | 月卡 ¥99 / 季卡 ¥267 |

#### F6: 项目管理
| 字段 | 说明 |
|------|------|
| 我的模型 | 列表展示，支持搜索 |
| 历史记录 | 保留 30 天 |
| 批量操作 | 删除、下载、重新生成 |

## 三、非功能需求

### 3.1 性能指标
| 指标 | 目标值 |
|------|--------|
| 首屏加载 | ≤2 秒 |
| 生成耗时 | 30-60 秒/个 |
| 并发支持 | 100 用户同时使用 |
| 可用性 | 99% (MVP 阶段) |

### 3.2 安全要求
- 用户上传内容加密存储
- API Key 服务端保管
- 防刷机制 (IP + 用户维度)
- 内容审核 (涉黄/涉政自动拦截)

### 3.3 兼容性
- 浏览器：Chrome 90+, Safari 14+, Edge 90+
- 移动端：iOS 14+, Android 10+
- 屏幕：响应式适配 (桌面 + 移动)
